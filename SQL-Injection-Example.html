Security Insight: SQL Injection

SQL Injection is a type of attack where an attacker inserts malicious SQL code into an application's input fields to manipulate the database query.

This works because the application directly inserts user input into an SQL query without validation or sanitisation.

Why It Happens

  • Unsanitised input
  • String‚Äëbuilt SQL queries
  • No use of parameterised statements

How To Prevent It

  • Use prepared statements / parameterised queries
  • Validate and sanitise all input
  • Use least-privilege database accounts
  • Avoid building SQL by string concatenation
